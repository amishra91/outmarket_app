# Outmarket_app

## Overview

This Flutter-based chat app.

## Features

- **Chat:** one-on-one chat capability.
- **History:** Display chat history for all users.

## Getting Started

### Installation

1. **Clone the Repository**

   ```bash
   clone the repository
   cd to the repo
   flutter pub get
   flutter run -d chrome
   This will launch the app in chrome.
   ```
